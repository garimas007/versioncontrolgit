Version control GIT -
A distributed version control system that allows multiple developers to work on a project simultaneously.
It tracks changes to files and directories over time, enabling collaboration and version management.

GIT - a software tool that implements version control using the GIT system.
GIT Hub - a web-based platform for hosting and managing GIT repositories.

git init - Initializes a new GIT repository in the current directory.
